# YouTube Clone

A responsive YouTube homepage clone built using HTML and CSS.

## Features

- Responsive video grid layout
- Fixed header and sidebar
- Search bar UI
- Tooltips
- Video duration overlays
- Flexbox and CSS Grid layouts
- Media Queries for responsiveness

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries



## Learning Outcomes

This project helped me practice:

- Semantic HTML
- Responsive Design
- CSS Positioning
- Flexbox
- CSS Grid
- UI Replication
